# koki-server
A node based chat server.

## install
```
$npm install -g koki-server
```

## start
```
$koki-server [PORT]
```

Make sure you have the port open on the firewall.

# For Client

Use nc to be a client.

eg.
```
$nc [HOST] [POST]
```

Create as many clients as you want.

Have fun chatting!!!
